 ## Shell Script
 사용자가 내린 명령을 운영체제가 수행할 수 있도록 명령을 입력받고,
 이를 컴퓨터의 운영체제에 전달함 
 
 순서 : 사용자 명령 -> shell 번역 -> 커널에서 shell 번역명령어 입력 -> 하드웨어 조작
 
 한국인 -> 번역가 -> 중국인     
 
 사용자 -> shell  -> 커널

## 명령어 

```pwd``` | 현재 경로
```cd``` | 폴더 이동
```mk``` | 파일 생성
```rm``` | 파일 삭제
```mkdir``` | 폴더 생성
```rmdir``` | 폴더 삭제
```cp``` | 복사

( file1을 file2로 복사함 ) | cd file1 folder/ ( file1를 folder/ 안에 복사함 ) | cd -r folder/  folderCopy/ ( folder를 folderCopy라는 이름으로 복사함 )
``` terminal
cp file1 file2 
cp file1 dir/
cp -r fileFolder dir/
```
